### Summary

Retrieve airport weather for a specific station

### Request

#### Code snippet: URL

`GET /v1/flight_operations/crew_services/COMMON_AIRPORT_WEATHER`
### Parameters

| Name | Type | Description | Required | Schema / Format |
| --- | --- | --- | --- | --- |
| station | Query | IATA station code | yes | xsd:string [A-Z]{3} |
| forTimeStamp | Query | If specified only return the forecast that is closest to the given time stamp, otherwise return all available forecasts. | no | xsd:dateTime |

### Responses

| Code | Description | Content-Type | Schema |
| --- | --- | --- | --- |
| 200 | Airport weather returned successfully | application/json | → [Common Weather Service](https://developer.lufthansa.com/docs/read/api_flightops/common_crew_services/Common_Weather_Service#commonweatherservice) Model |
| 409 | Invalid parameters, e.g. missing airport code | application/json | → [Processing Errors](https://developer.lufthansa.com/docs/api_flightops/common_crew_services/Processing_Errors_Model) Model |
| 500 | airport does not exist or no weather available | application/json | → [Processing Errors](https://developer.lufthansa.com/docs/api_flightops/common_crew_services/Processing_Errors_Model) Model |
