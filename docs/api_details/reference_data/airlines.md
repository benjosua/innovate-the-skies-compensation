Retrieves the complete details of one particular airline or list of airlines.

### Request URIs

#### Code snippet: Call

```
GET /mds-references/airlines
GET /mds-references/airlines/{airlineCode}
GET /mds-references/airlines[?limit={recordLimit}][&][offset={recordOffset}]
```

| Variable | Description | Format |
| --- | --- | --- |
| _{airlineCode}_ | Optionally return only this airline. | 2-character IATA airline designator code, e.g. “LH”. |
| _{recordLimit}_ | Optionally sets the number of records returned. Defaults to 20, maximum is 100. | integer, e, g, 44 |
| _{recordOffset}_ | Optionally sets the number of records skipped when sorting response records alphabetically. Defaults to 0. | integer, e.g. 123 |

### Request Examples

#### Code snippet: Call

```
GET /mds-references/airlines
GET /mds-references/airlines/UA
GET /mds-references/airlines?limit=44&offset=123
```

### Response Structure Definition

| Key | Description |
| --- | --- |
| AirlineResource | Root element of airlines response. |
| .Airlines | Container for airline elements. |
| ..Airline[] | Array of all available airlines or one airline matching the request. |
| ...AirlineID | 2-character IATA airline code, e.g. “LH”. |
| ...AirlineID_ICAO | 3-letter ICAO airline code, e.g. "DLH". |
| ...Names | Container for airline full names. |
| ....Name[] | Array: language specific airline name, e.g. “Lufthansa”. |
| @LanguageCode | 2-letter ISO 639-1 language code for the corresponding item. |
| .Meta | Container for meta links. |
| ..Link[] |  |
| @Href | Link to actual a resource. |
| @Rel | Specifying kind of link such as ‘self’ (link that returned this response), ‘alternate’ (link that points to another resource) or ‘related’ (link that points to related resource). |

### Response Example

```xml
<?xml version="1.0" encoding="UTF-8"?>
<AirlineResource>
  <Airlines>
```
